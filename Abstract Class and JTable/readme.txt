You'll Learn: 
How to add JTable(DefaultTableModel) to Java Swing UI (Jframe).
And Abstract Classes, ComboBox, Vector.



Task:
The step you've to follow here is to first populate the ComboBox &
then specify the column names of the table & then provide handler 
for the  “Add Data in Table” button.  
For item selected in the ComboBox, create 3 objects & 
populate the table with the name of object, their dimension &
the area of the object. 
The ComboBox handler retries the item selected from the ComboBox.
